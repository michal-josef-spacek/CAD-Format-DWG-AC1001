=head1 NAME

CAD::Format::DWG::AC1001 - Read AutoCAD r2.4 DWG file.

=head1 SYNOPSIS

 use CAD::Format::DWG::AC1001;

 my $dwg = CAD::Format::DWG::AC1001->new;
 my $dwg = CAD::Format::DWG::AC1001::from_file($dwg_file);
 my $blocks_ar = $dwg->blocks;
 my $entities = $dwg->entities;
 my $header = $dwg->header;
 my $layers_ar = $dwg->layers;
 my $linetypes_ar = $dwg->linetypes;
 my $styles_ar = $dwg->styles;
 my $views_ar = $dwg->views;
 my $block_entities = $dwg->block_entities;

=head1 DESCRIPTION

In distribution is kaitai structure definition file dwg_ac1001.ksy which
described AutoCAD DWG format version AC1001.

=head1 DEPENDENCIES

L<IO::KaitaiStruct>.
